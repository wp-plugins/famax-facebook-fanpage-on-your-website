== Description ==

Famax will pull Posts from any Facebook FanPage and display them on your website. 

Check out the <a href="http://buildwithcards.com/chwp/famaxfun/" target="_blank">WordPress Demo here</a>.

You can also see how your FanPage looks like with Famax at the <a href="http://demos.codehandling.com/famax/famax.html" target="_blank">Live Edit Demo</a>.

Refer here for <a href="http://www.codehandling.com/2015/03/free-version-of-famax-wordpress.html" target="_blank">complete documentation</a> of the plugin.

If you want to display your FanPage Albums, Tags, Videos etc. with Famax, check out the <a href="http://codecanyon.net/item/famax-facebook-fanpage-albums-on-your-website/10429010" target="_blank">Premium version</a>!

== Installation ==

1. Upload 'famaxfun-wordpress' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add [famaxfun] shortcode in the pages where you want your fanpage to appear.

Check the <a href="http://www.codehandling.com/2015/03/free-version-of-famax-wordpress.html" target="_blank">detailed instructions and Shortcode attributes</a>.
